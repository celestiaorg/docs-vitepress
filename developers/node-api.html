blob

Get

blob.Get( height uint64, nID namespace.ID, commitment blob.Commitment )*blob.Blob
perms: read

Get retrieves the blob by commitment under the given namespace and height.

Request

{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "blob.Get",
  "params": [
    42,
    "Bw==",
    "Bw=="
  ]
}

Response

{
  "id": 1,
  "jsonrpc": "2.0",
  "result": [
    {
      "namespace": "KmJ5dGUgYXJyYXk=",
      "data": "Ynl0ZSBhcnJheQ==",
      "share_version": 42,
      "commitment": "Bw=="
    }
  ]
}

Included

blob.Included( height uint64, nID namespace.ID, proof *blob.Proof, commitment blob.Commitment )bool
perms: read

Included checks whether a blob's given commitment(Merkle subtree root) is included at given height and under the namespace.

Request

{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "blob.Included",
  "params": [
    42,
    "Bw==",
    [
      {
        "start": 0,
        "end": 4,
        "nodes": [
          "dGVzdA=="
        ]
      }
    ],
    "Bw=="
  ]
}

Response

{
  "id": 1,
  "jsonrpc": "2.0",
  "result": [
    true
  ]
}

Submit

blob.Submit( blobs []*blob.Blob )uint64
perms: write

Submit sends Blobs and reports the height in which they were included. Allows sending multiple Blobs atomically synchronously. Uses default wallet registered on the Node.

Request

{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "blob.Submit",
  "params": [
    [
      {
        "namespace": "KmJ5dGUgYXJyYXk=",
        "data": "Ynl0ZSBhcnJheQ==",
        "share_version": 42,
        "commitment": "Bw=="
      }
    ]
  ]
}

Response

{
  "id": 1,
  "jsonrpc": "2.0",
  "result": [
    42
  ]
}
